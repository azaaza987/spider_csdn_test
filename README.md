# spider_csdn
Python3 抓取csdn博客访问量生成数据,并且读取数据绘图

CSDN博客教程地址：https://blog.csdn.net/s740556472/article/details/78239204

csdn_spider 本项环境目须知:

1.系统环境:Windows10

2.语言环境:Python3.6

3.集成库环境:安装好python的Beatifulsoup4,requests,lxml,matplotlib库
